Запуск шардированного клика
из этой директории
``` bash
docker-compose up
```
Логин и пароль - user password. Меняется в файле users.xml.
Имя контейнера шарда clickhouse-shard-n, где n номер шарда, напр. clickhouse-shard-1. Внешние порты tcp 4160n и http 4161n. Меняется в compose файле.
Имя группы контейнеров "new". Меняется в compose файле, лучше поменять на имя той группы, в которой должны быть шарды клика.
Так же не забыть поменять сеть ch_sharded_net на нужную в группе контейнеров, что бы видеть другие контейнеры в группе.
